# Blog Template

This is a **Template Repository** used by the Content Publisher Agent.
When a new blog category is detected, the agent clones this repository to create a new blog.

## Features
- Jekyll Minimal Theme
- SEO Tag Plugin
- **Google AdSense Ready** (Edit `_layouts/default.html`)

## How to use
1. Go to `_layouts/default.html` and uncomment the AdSense script block.
2. Add your AdSense Client ID.
